## 🔧 技術棧

### 當前使用
- **語言**: Python 3.11
- **容器**: Docker, Docker Compose
- **API**: GitHub REST API v3
- **通知**: Slack Webhooks
- **調度**: schedule 庫
- **配置**: YAML, dotenv

### 計劃採用
- **Web**: Flask/FastAPI
- **前端**: React/Vue.js
- **資料庫**: PostgreSQL
- **快取**: Redis
- **監控**: Prometheus + Grafana
- **日誌**: ELK Stack

- **記憶體使用**: ~100MB
